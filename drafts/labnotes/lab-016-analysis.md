## 分析スクリプト

```python
import json, math, sys, statistics
sys.path.insert(0, '.')
from signal_lab_verify import closed, win, match, compute, wilson, get_trend, GROUPS, REV

with open("signals-log.json", encoding="utf-8-sig") as f:
    raw = json.load(f)
data = raw

def stats(data, filt):
    k, n = compute(data, filt)
    ci_lo, ci_hi = wilson(k, n)
    if n == 0: return k, n, 0, 0, 0, 0, 0
    rs = [1.33 if win(d) else -1.0
          for d in data if closed(d) and match(d, filt)]
    mean_r = sum(rs)/len(rs) if rs else 0
    if len(rs) > 1:
        sd = statistics.stdev(rs)
        se = sd/math.sqrt(len(rs))
        r_lo, r_hi = mean_r-1.96*se, mean_r+1.96*se
    else:
        r_lo = r_hi = mean_r
    return k, n, ci_lo, ci_hi, mean_r, r_lo, r_hi
```

## 交絡点検
1. **上昇×ロング(9.8%)は交絡か**：N=41で「上昇相場でロング」なのに9.8%は驚異的に低い。
   - 他グループとの比較：index×long 53.7%、jpy_fx×long 40.6%。同じロングでも other_fx のみ異常値
   - 「上昇に乗ろうとしてロングしているはずなのに負ける」→ other_fx の上昇は「ドル強含み」期間が多く、逆張り系シグナルが多発している可能性
2. **下降×ロング(55.6%)の逆説**：N=54で55.6%はCI下限42.4%≥43%に近い
   - ただしCI幅広く[42.4%~68.0%]、E(R)=+0.294もCI[-0.017~+0.606]（0跨ぎ）→確定打なし
3. **AUDUSD=X例外**: 19/39=48.7% で他と比べ高い。AUDは中国連動（#009でも登場）
4. **シグナル交絡**: signal=rsi_oversold_bounce のみ41.7%（N=36）と相対的にマシだが全て損益分岐以下
